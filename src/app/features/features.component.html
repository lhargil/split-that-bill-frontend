Features

Split that bill offers an intuitive interface that allows users to enter the bill details, bill items, friends as well as extra charges. This includes a stepped wizard interface, short but concise forms, modals, dialogs and notifications. Textual cues in each wizard step are in place to give users a clear indication of where they are in the flow. All these are provided to enable the user a hassle-free experience in splitting bills.

Bill assignment

The bill assignment step is one of the most important features of the app. This enables the user to assignment a bill item to a friend. As soon as this assignment is done, the calculation for the payable is automatically engaged in the background. The app promotes transparency by displaying the friend's total payable when extra charges like taxes and service charges as well as item discounts are included.


Auto-calculation

Unit prices, extra charges and discounts all contribute to the total payable amount. When these are specified dutifully, the amount updates just as easily.


Receipt

A receipt is created once the bill details and assignments are completed. Once saved, the user is then redirected to the receipt page that displays all the pertinent details. It is a publicly-accessible page, in which the url to it can easily be shared to the friends involved.
