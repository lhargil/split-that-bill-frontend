Features

Split that bill offers an intuitive interface that allows users to enter the bill details, bill items, friends involved as well as extra charges with ease. This includes a stepped wizard interface, short but concise forms, modals, dialogs and notifications. Textual cues in each step are in place to help guide the user as they go, enabling them a hassle-free experience in splitting bills.

Bill assignment

The bill assignment is one of the most important features of the app. This step lets the user match an item up to a friend. Friends can be assigned to multiple items.

Auto-calculation

Unit prices, extra charges and discounts all contribute to the total payable amount. When these are specified dutifully, the amounts update just as easily.

Receipt

A receipt is created once the bill details and assignments are completed. Once saved, the user is then redirected to the receipt page that displays all the pertinent details about the split. It is a publicly-accessible page, the url to which can readily be shared to the friends involved.
